# uneZpa-ol
Es un intérprete de expresiones regulares a python (it's an interpreter that converts regular expressions to Python language).
Permite:
Declaración de variables (entero, cadena)
Asignación mediante expresiones
Condicionales Si ... Entonces ... FinSi
Bucles Mientras ... Hacer ... FinMientras
Su estructura es:
/
├── UNEZPANOL.py
├── README.md
├── .gitignore
├── .github/
│   └── workflows/
│       └── ci.yml
└── tests/
│   └── test1.py
└── evidencias
    
Evidencias necesarias se encuentran en la carpeta /evidencias:
Pipeline ejecutándose en GitHub Actions
Resultado de pylint
Resultado de mypy
Pruebas pytest + coverage
